Recursos Principais

Consulta de Disciplinas

Encontre as disciplinas disponíveis em cada curso. Esta funcionalidade permite visualizar as disciplinas oferecidas em cada curso, bem como informações detalhadas sobre cada uma delas, como horários, professores responsáveis, etc.

Horários do Professor

Veja os horários de aula de cada professor. Com esta funcionalidade, é possível acessar os horários de aula de cada professor, facilitando a organização e planejamento das atividades acadêmicas.

Consulta de Alunos

Visualize os alunos matriculados em uma disciplina. Esta funcionalidade permite visualizar informações detalhadas sobre os alunos matriculados em uma disciplina específica, incluindo dados pessoais, notas, frequência, entre outros.

Consultar Cursos

Obtenha informações sobre cursos e seus coordenadores. Com esta funcionalidade, é possível obter informações detalhadas sobre os cursos oferecidos pela instituição, incluindo ementas, carga horária, requisitos, entre outros.

Registro de Interesse

Registre seu interesse em lecionar uma disciplina. Esta funcionalidade permite que os professores expressem seu interesse em lecionar disciplinas específicas, facilitando o processo de alocação de professores e distribuição de disciplinas.

Lista Trabalhos

Visualize a lista de trabalhos atribuídos às disciplinas. Esta funcionalidade permite que os professores vejam uma lista detalhada dos trabalhos atribuídos a cada disciplina, incluindo a data de entrega, contexto do trabalho e informações sobre os alunos envolvidos.
